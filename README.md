# Ganesh Surgical Store

Frontend-only e-commerce website built with HTML5, CSS3, and vanilla JavaScript.

## Pages

- Home
- Products
- Product Details
- Categories
- About Us
- Contact Us
- Login
- Register
- Profile
- Cart
- Wishlist
- Orders
- 404

## Features

- Local Storage register, login, logout, remember user, profile editing, and profile image upload
- Local Storage cart, wishlist, contact messages, and order history
- 20 surgical and medical products with local SVG images
- Live search suggestions, category filtering, and price/rating sorting
- Floating offline chatbot with product, shipping, contact, and category help
- Fully static and compatible with GitHub Pages

## Run

Open `index.html` directly in a browser or publish this folder with GitHub Pages.
